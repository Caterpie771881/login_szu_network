# login_szu_network

一个用于在命令行环境下登陆深大校园网的脚本

效果展示:

![example](./example.png)

# 使用说明

目前该项目仅支持 linux 系统

## 下载项目

```
git clone https://github.com/Caterpie771881/login_szu_network.git
```

## 给脚本赋予执行权限

```
cd login_szu_network
chmod +x login_szu_network.sh
```

## 使用 bash 执行脚本
```
bash login_szu_network.sh
```
